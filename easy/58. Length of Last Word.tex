给定一个仅包含大小写字母和空格 ' ' 的字符串，返回其最后一个单词的长度。

如果不存在最后一个单词，请返回 0 。

说明：一个单词是指由字母组成，但不包含任何空格的字符串。

示例:

输入: "Hello World"
输出: 5



























思路：
使用 istringstream in(s);






















code：
class Solution {
public:
    int lengthOfLastWord(string s) {
        vector<string> res;
        istringstream in(s);
        string t;
        while(in >> t)
            res.push_back(t);
        if(res.size() == 0) return 0;
        return res[res.size()-1].length();
    }
};
