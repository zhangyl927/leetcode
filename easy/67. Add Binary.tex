给定两个二进制字符串，返回他们的和（用二进制表示）。

输入为非空字符串且只包含数字 1 和 0。

示例 1:

输入: a = "11", b = "1"
输出: "100"

示例 2:

输入: a = "1010", b = "1011"
输出: "10101"





























思路：
从最后一位往前加，每次计算的结果 （c % 2 + '0'） 加上 s 即为当前结果。
当 a 移到首位， b 移到首位，且进位 c == 0 时，结束。

























code：
class Solution {
public:
    string addBinary(string a, string b) {
        string s = "";
        int c = 0, i = a.size()-1, j = b.size()-1;
        while(i >= 0 || j >= 0 || c == 1)
        {
            c += (i >= 0) ? a[i--] - '0' : 0;
            c += (j >= 0) ? b[j--] - '0' : 0;
            s = char(c%2 + '0') + s;
            c /= 2;
        }
        return s;
    }
};
