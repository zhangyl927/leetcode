给定一个正整数，检查他是否为交替位二进制数：换句话说，就是他的二进制数相邻的两个位数永不相等。

示例 1:

输入: 5
输出: True
解释:
5的二进制数是: 101
示例 2:

输入: 7
输出: False
解释:
7的二进制数是: 111
示例 3:

输入: 11
输出: False
解释:
11的二进制数是: 1011
 示例 4:

输入: 10
输出: True
解释:
10的二进制数是: 1010



























思路：
temp 存放末位一位二进制，该数右移一位，比较最末位是否等于 temp，等于返回 false，不等于则 temp 存储当前最末位；



























code：
class Solution {
public:
    bool hasAlternatingBits(int n) {
        int temp = n%2;
        while(n)
        {
            n >>= 1;
            if(n%2 == temp) return false;
            temp = n%2;
        }
        return n == temp;
    }
};
