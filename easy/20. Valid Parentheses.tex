给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"
输出: true
示例 2:

输入: "()[]{}"
输出: true
示例 3:

输入: "(]"
输出: false
示例 4:

输入: "([)]"
输出: false
示例 5:

输入: "{[]}"
输出: true







思路：
使用栈，最后栈为空则有效。














code：
class Solution {
public:
    stack<char> myStack;
    bool isValid(string s) {
        if(s == "") return true;
        for(int i = 0; i < s.size(); i++)
        {
            if(myStack.empty() && (s[i] == ')' || s[i] == ']' || s[i] == '}')) 
                return false;
            if(s[i] == '(' || s[i] == '{' || s[i] == '[')
                myStack.push(s[i]);
            else if(myStack.top() == '(' && s[i] == ')')
                myStack.pop();
            else if(myStack.top() == '[' && s[i] == ']')
                myStack.pop();
            else if(myStack.top() == '{' && s[i] == '}')
                myStack.pop();
            else return false;
        }
        return myStack.empty();
    }
};
