统计所有小于非负整数 n 的质数的数量。

示例:

输入: 10
输出: 4
解释: 小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。










思路：
统计 1 - sqrt(n) 之间的数即可。
优化：
瓜拉多塞 质数筛法。














code：
class Solution {
public:
    int countPrimes(int n) {
        int result = 1;
        if(n < 3) return 0;
        else
        {
            for(int i = 3; i < n; i++)
            {
                if(i % 2 == 0)
                {
                    continue;
                }
                bool flag = true;
                for(int j = 3; j <= sqrt(i); j += 2)
                {
                    if(i % j == 0)
                    {
                        flag = false;
                        break;
                    }
                }
                if(flag)
                    result++;
            }
            return result;
        }
    }
};
