给定一个正整数，返回它在 Excel 表中相对应的列名称。

例如，

    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...
示例 1:

输入: 1
输出: "A"
示例 2:

输入: 28
输出: "AB"
示例 3:

输入: 701
输出: "ZY"



























思路：
对 26 取余，结果转换为字符类型保存在字符串中。最后一位最先保存在字符串中，所以需对字符串进行反转。



























code:
class Solution {
public:
    string convertToTitle(int n) {
        string s = "";
        while(n)
        {
            n--;
            s += 65 + n%26;
            n = n/26;
        }
        reverse(s.begin(), s.end());
        return s;
    }
};
